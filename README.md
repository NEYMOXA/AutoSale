# AutoSale
## Getting Started

1. Clone the repository:
```bash
git clone https://github.com/yourusername/autosale-dealership.git
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Build for production:
```bash
npm run build
```

## Technologies Used

- HTML5
- SCSS
- Modern CSS Grid and Flexbox
